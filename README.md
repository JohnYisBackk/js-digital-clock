# ⏰ Digital Clock PRO

A modern digital clock built with **HTML, CSS and vanilla JavaScript**.
The clock displays the current **time, day of the week and full date** and updates automatically every second.

## ⚙️ How It Works

JavaScript creates a new `Date` object and extracts:

* hours
* minutes
* seconds
* day
* month
* year

The clock updates every second using:

```
setInterval(updateClock, 1000);
```
